# Experimentos de fairness

Esta pasta será usada para experimentos sobre avaliação de vieses, impactos desiguais e métricas de fairness.

Experimentos nesta área devem explicitar grupos analisados, variáveis usadas, métricas escolhidas, limitações estatísticas e riscos de interpretação. Resultados não devem ser usados como garantia de justiça; eles são evidências para análise contextual.
