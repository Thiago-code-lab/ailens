# Checklists

A pasta `checklists/` reúne listas práticas para revisão de sistemas, documentos, fluxos e decisões envolvendo IA.

Checklists não substituem julgamento técnico, jurídico, regulatório ou de segurança. Eles funcionam como apoio para reduzir esquecimentos, padronizar análise e facilitar discussões entre áreas.

## Como usar

1. Defina o sistema, produto ou experimento analisado.
2. Responda cada item com evidência, não apenas com opinião.
3. Registre lacunas e responsáveis.
4. Revise os itens quando o sistema mudar.
5. Use a matriz de risco quando uma resposta indicar impacto relevante.
