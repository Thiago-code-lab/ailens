# Modelos de documentação

A pasta `templates/` reúne modelos de documentação para sistemas de IA. Eles devem ser adaptados ao contexto, ao risco e ao nível de maturidade de cada projeto.

## Modelos disponíveis

- `model-card.md`: ficha para documentar objetivo, dados, métricas, limitações, riscos, fairness, explainability e monitoramento de um modelo ou sistema.
- `matriz-risco-ia.md`: estrutura para registrar riscos, impacto, probabilidade, severidade, controles, evidências e responsáveis.

## Uso recomendado

Preencha os modelos durante o desenvolvimento, não apenas no fim do projeto. Quando uma informação ainda não existir, registre a lacuna, o responsável e a data de revisão esperada.
